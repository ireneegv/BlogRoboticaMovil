# PRACTICA 5
----

En esta práctica se ha creado un mapa a partir de las detecciones del láser y de la posición de odom. También se han implementado la actualización con la regla de Bayes para representar los datos obtenidos por el láser, actualizando de forma incremental la probabilidad de que cada celda esté ocupada o libre según las mediciones. Para el movimiento se utilizó un desplazamiento pseudoaleatorio, debido a esto, como se observa en el video, tarda mucho tiempo en llegar a cubrir todas las zonas.

----
# DIFICULTADES
----

A la hora de representar en el mapa los datos del láser surgieron bastantes dificultades debido a que este estaba rotado. Una vez aplicadas las conversiones adecuadas y seleccionados correctamente los rayos del láser, ya se pudo emplear el ángulo de manera normal y aplicar la transformación necesaria para que se mostrara en la posición correcta.
